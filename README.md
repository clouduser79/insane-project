# Image Slideshow Creator

A beautiful and interactive web application that transforms your images into stunning slideshows with dynamic backgrounds and music.

## Features

- **Image Upload**: Upload multiple images to create your slideshow
- **Music Integration**: Choose from a selection of background music
- **Dynamic Backgrounds**: Beautiful gradient transitions and sparkle effects
- **Responsive Design**: Works on desktop and mobile devices
- **Customizable**: Add a name to your presentation
- **Visual Effects**: Smooth transitions and animations

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: CSS3 with modern animations
- **Audio**: Web Audio API
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (v16 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/clouduser79/insane-project.git
   cd insane-project/frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## How to Use

1. Enter a name for your presentation
2. Upload your images (5 or more recommended)
3. Select background music from the dropdown
4. Click "Start Presentation" to begin
5. Use the volume control to adjust audio levels
